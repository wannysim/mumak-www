## A general Lightroom edit order

You said you start from "darks → shadows," but the broader pattern goes **macro → micro**.

1. **WB (Temperature / Tint)** — set the color baseline
2. **Exposure** — overall brightness
3. **Highlights / Shadows / Whites / Blacks** — recover by tonal region
4. **Tone Curve** — fine-tune what sliders couldn't
5. **HSL / Color Mixer** — isolate specific colors
6. **Sharpening / Noise Reduction** — detail
7. **Crop / Spot Removal** — final cleanup

> **Principle**: big decisions first, small ones later. If WB is off and you start tweaking HSL, everything will need redoing.

## Easy points of confusion

- **Lightroom's K slider runs opposite to physical Kelvin.** See [[white-balance]].
- **Highlights and Whites are not the same thing.** See [[dynamic-range]].
- **JPEG's editing limit is set by bit depth.** See [[raw-vs-jpeg]].
- **An S-curve also raises saturation, not just contrast.** See [[tone-curve]].
