사진의 **흰색을 진짜 흰색으로 보이게** 만드는 작업. 광원의 색을 카메라가 중성화하는 과정.

## 색온도(K)의 두 얼굴

### 물리적 색온도

흑체복사 이론: 물체를 가열할수록 방출하는 빛이 **빨강 → 주황 → 노랑 → 흰색 → 파랑** 순으로 변한다.

| 광원             | 색온도(K)    | 보이는 색 |
| ---------------- | ------------ | --------- |
| 촛불             | ~1800K       | 진한 주황 |
| 백열등           | ~3000K       | 따뜻      |
| 형광등(주백색)   | ~4000K       | 중성      |
| 정오 일광        | ~5500K       | 흰색      |
| 흐린 날          | ~6500K       | 약간 푸른 |
| 그늘 / 푸른 시간 | ~8000-10000K | 푸른      |

여기까지는 **K가 높을수록 차가운(푸른) 빛**이 맞다.

### 🔴 Lightroom 슬라이더는 반대로 작동한다

> Lightroom의 **Temperature 슬라이더를 오른쪽(높은 K)으로 끌면 사진이 따뜻해진다.**

이유: 슬라이더는 **광원의 색온도를 명시하는 값**이다. "이 사진은 6500K 광원에서 찍혔다"고 알려주면, Lightroom은 그 푸른 빛을 **상쇄하기 위해 따뜻한 보정을 더한다**.

```
실제 광원      Lightroom의 보정         결과
─────────      ──────────────          ────
3000K (따뜻)  → 슬라이더 3000K → 푸르게 보정 → 사진은 중성
5500K (중성)  → 슬라이더 5500K → 보정 거의 없음 → 사진은 중성
8000K (차가움) → 슬라이더 8000K → 따뜻하게 보정 → 사진은 중성
```

요약하면: **물리 K↑ = 차가운 빛**, **슬라이더 K↑ = 따뜻한 결과**. 직관과 반대로 보이는 진짜 이유.

## Tint — 녹색 ↔ 마젠타 축

색온도는 빨강↔파랑 축만 다룬다. 하지만 실제 광원은 한 축으로 안 잡힌다 — 특히 형광등·LED는 녹색이 끼고, 일몰 직후는 마젠타가 낀다.

> **Tint = 녹색-마젠타 축의 보정**.
> 슬라이더 왼쪽(녹색) ↔ 오른쪽(마젠타).

광원별 흔한 Tint 방향:

| 광원             | 보정 방향                      |
| ---------------- | ------------------------------ |
| 형광등           | **마젠타 쪽으로** (녹색 보정)  |
| 일출/일몰        | 녹색 쪽으로 살짝 (마젠타 억제) |
| 푸른 시간 / 그늘 | 마젠타 쪽으로 살짝             |
| LED 저가 광원    | 케이스별 — 보통 마젠타         |

Tint가 어렵게 느껴지는 이유는 **얼굴이 약간 마젠타/녹색 쪽으로 치우치는 걸 인지하기 어렵기 때문**. 노하우:

- 흰 종이 / 흰 벽이 화면에 있는 사진을 기준으로 잡는다.
- **인물의 피부**가 노란 듯 녹색 쪽이면 → Tint를 마젠타 쪽으로.
- 햇빛 + 그늘이 섞인 사진은 **얼굴 위주로 맞추고 배경은 포기**.

## 실전: AWB로 찍을 때의 함정

본인 워크플로우(AWB + RAW). 장점은 셔터 누를 때 신경 끄기. 단점:

- **연속 컷 사이에 WB가 미세하게 흔들린다.** 같은 장면 여러 컷 보정할 때 컷마다 다른 K값이 나온다.
- **색이 강한 피사체가 화면에 들어오면 AWB가 끌려간다.** 빨간 옷이 큰 비중이면 카메라가 "장면이 너무 빨갛다"고 판단해 시안 쪽으로 보정 → 얼굴이 시퍼래짐.

대처: Lightroom에서 한 컷의 WB를 잡고 **나머지에 동기화(Sync)**. 같은 장면 묶음마다 한 번씩.

## Lightroom의 WB Picker (스포이드)

화면 안에서 **회색이어야 할 영역**을 클릭하면 자동으로 WB를 맞춰준다. 인물 사진이라면 **흰자위 / 회색 셔츠 / 회색 도로** 같은 무채색 영역이 잘 동작.

색이 있는 영역(피부, 잎, 옷)을 찍으면 그 색을 무채색으로 만들려 해서 사진 전체가 망가짐.
