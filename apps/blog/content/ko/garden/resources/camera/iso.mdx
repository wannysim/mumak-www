빛에 대한 **감도**. 정확히 말하면 디지털 카메라에서 ISO는 **들어온 신호를 얼마나 증폭할지**의 값이다.

- 낮은 ISO (100, 200): 신호 증폭 적음 → 깨끗하지만 빛이 많이 필요
- 높은 ISO (3200, 12800): 신호 증폭 많음 → 어두운 곳에서 찍을 수 있지만 **노이즈/그레인** 같이 증폭됨

## "감도"라는 말의 함정

필름 시대엔 ISO가 진짜로 필름의 화학적 감도였다. 디지털에선 **센서의 물리 감도는 고정**이고, 그 위에 신호 처리 단계에서 증폭(gain)을 얼마나 줄지의 문제다.

> 그래서 **고 ISO로 찍는 것 ≈ 저 ISO로 찍은 뒤 후보정에서 노출을 올리는 것**에 가깝다 (ISO-invariant 센서일 때).

## 노이즈가 생기는 이유

신호를 키우면 신호와 함께 **노이즈도 키워진다**. 어두운 곳일수록 원본 신호가 작아서, 증폭하면 노이즈 비율(SNR)이 더 두드러진다.

## Base ISO

센서가 **가장 깨끗하게 동작하는 네이티브 감도**. 보통 ISO 100 또는 200.

- Base ISO에서 다이내믹 레인지가 가장 넓다
- 그 아래 (예: ISO 50)는 "확장 ISO"인 경우가 많음 → 오히려 하이라이트가 잘림

## Dual Native ISO

일부 카메라는 센서 회로에 **네이티브 감도가 두 개** 있다. 임계 ISO를 넘으면 두 번째 회로로 전환되며 **노이즈가 다시 줄어든다**.

| 카메라         | Native ISO 1 | Native ISO 2 |
| -------------- | ------------ | ------------ |
| Sony FX3 / FX6 | 800          | 12800        |
| Panasonic GH5s | 400          | 2500         |
| Canon C70      | 800          | 4000         |

→ 저조도 영상에서 **ISO 6400보다 ISO 12800이 깨끗할 수도 있다**는 비직관적 현상이 일어난다. Dual Native가 있는 카메라라면 두 번째 베이스를 적극적으로 쓰는 것이 유리.

## 실전 휴리스틱

- **밝은 곳**: 무조건 base ISO 고정
- **저조도 사진**: 손떨림 한계까지 셔터를 늘리고, 조리개 개방, 그래도 모자라면 ISO
- **저조도 영상**: 셔터는 못 건드리니 → 조리개 개방 → ISO를 native 2로 점프
