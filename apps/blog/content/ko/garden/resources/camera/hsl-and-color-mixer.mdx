색을 **8개 채널**(Red / Orange / Yellow / Green / Aqua / Blue / Purple / Magenta)로 분리해서 따로 만지는 패널. 본인이 잘 쓸 줄 모른다고 했으니 여기서부터.

## 세 슬라이더의 차이

각 채널마다 세 슬라이더가 있다.

| 슬라이더              | 만지는 것       | 직관적 효과                       |
| --------------------- | --------------- | --------------------------------- |
| **Hue (색상)**        | 그 색의 톤 자체 | 빨강을 더 주황으로, 또는 자홍으로 |
| **Saturation (채도)** | 그 색의 진하기  | 회색에 가까이 ↔ 진하게            |
| **Luminance (휘도)**  | 그 색의 밝기    | 그 색만 어둡게/밝게               |

> 핵심: **다른 색은 건드리지 않고 한 색만 따로** 조정할 수 있다.

## 가장 많이 쓰는 4가지 패턴

### 1. 인물 피부 다듬기 — Orange 채널

대부분의 인종에서 **피부톤은 Orange 채널**에 들어간다 (약간의 Red 포함).

- **Orange Saturation -10**: 피부 채도를 살짝 빼서 **자연스러운 톤**. 과한 적기 / 화장 느낌 줄이기.
- **Orange Luminance +10**: 피부를 **밝고 깨끗하게**. 잡티 인상이 줄어듦.
- **Orange Hue 약간 -**: 너무 노랗게 나온 피부를 **분홍 쪽으로**.

이거 하나만 익혀도 인물 사진이 달라진다.

### 2. 하늘 살리기 — Blue / Aqua 채널

- **Blue Saturation +**: 하늘이 더 깊은 파랑.
- **Blue Luminance -**: 하늘이 더 진해짐 (구름과의 대비 증가).
- **Aqua**도 만지는 이유: 한국 하늘은 순수 Blue보다 **Aqua가 섞여** 있어서.

### 3. 잎사귀 / 풀 톤 — Yellow / Green 채널

- 카메라가 잎을 너무 노랗게 잡을 때 **Green Hue를 -** (더 시원한 녹색으로).
- **Yellow Saturation -**로 누런 잔디를 누르고, **Yellow Luminance +**로 가을 단풍을 살림.

### 4. 옷 / 사물 색 보정

빨간 옷을 **Magenta 쪽으로** 살짝 (Red Hue -) 옮기면 더 고급스럽게 보이고, **Saturation을 살짝 빼면** 전체 톤과 어울린다.

## 본인의 시그니처 톤 만들기

자주 만지는 채널 한두 개를 정해두면 본인 톤이 만들어진다. 예시:

| 톤 컨셉            | 자주 만지는 것                                                  |
| ------------------ | --------------------------------------------------------------- |
| 깨끗한 일상        | Orange Lum +, Yellow Sat -, Blue Lum -                          |
| 빈티지 / 필름      | Orange Sat -, Green Hue +, Yellow Hue +                         |
| 시네마틱 / 진한    | Blue Sat +, Orange Sat +, Teal & Orange 커브 (→ [[tone-curve]]) |
| 미니멀 / 채도 낮음 | 전체 Sat -10~-15, Luminance만 미세 조정                         |

## Targeted Adjustment Tool (TAT)

Lightroom 패널 좌상단의 **작은 동그라미 아이콘**. 사진 위 한 점을 클릭하고 **드래그**하면 그 영역의 채널을 자동으로 인식해 슬라이더가 움직임.

- 인물 얼굴을 잡고 위로 드래그 → Orange Luminance가 자동으로 올라감
- 하늘을 잡고 아래로 드래그 → Blue Luminance가 내려감 (어두운 하늘)

색 채널을 외울 필요 없이 **눈으로 보면서 만지는 방법**. HSL을 처음 배우는 사람은 이걸로 시작하는 게 빠르다.

## 자주 빠지는 함정

- **너무 많이 만지지 않기**. 슬라이더 ±10 안에서 거의 모든 효과가 가능. ±50을 넘기면 색이 깨지거나 마스크 경계가 보임.
- **Saturation과 Vibrance는 다르다**. Vibrance는 "이미 채도 낮은 색만 더 끌어올리는" 보정. 인물 사진에서 피부 보호용.
- **HSL은 [[white-balance|화이트밸런스]] 다음에**. WB가 안 맞은 상태에서 만지면 다음에 다 흔들린다.
