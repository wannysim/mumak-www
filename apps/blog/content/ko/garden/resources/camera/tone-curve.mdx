밝기 입력을 출력으로 매핑하는 그래프. 슬라이더로 못하는 **세밀한 영역별 조정**과 **시그니처 톤 만들기**의 핵심 도구.

## 그래프 읽는 법

```
출력
↑ 1.0  ┌────────┐
       │       ╱
       │      ╱  ← 이 선을 어떻게 휘게 하느냐가 전부
       │     ╱
  0.5  │    ╱
       │   ╱
       │  ╱
       │ ╱
  0.0  └─────────→ 입력
       0    0.5   1.0
       (어두움)  (밝음)
```

- **X축(입력)**: 원본 사진의 밝기
- **Y축(출력)**: 보정 후 밝기
- **기본 직선**: "원본 그대로". 직선을 휘면 톤이 변한다.

곡선 위 한 점을 위로 올리면 **그 입력 밝기에 해당하는 영역이 더 밝아짐**. 아래로 내리면 어두워짐.

## 4개 영역으로 사고하기

Lightroom 톤 커브는 자동으로 영역을 나눠준다:

```
0 ──── 25 ──── 50 ──── 75 ──── 100
 Blacks   Shadows   Lights   Highlights
   ↑        ↑        ↑          ↑
 검정 끝   어두운   밝은      흰색 끝
          중간톤   중간톤
```

슬라이더의 Blacks/Shadows/Highlights/Whites와 거의 같지만, **커브는 영역의 경계를 본인이 정할 수 있다**.

## S 커브 — 본인이 자주 쓰는 그것

```
출력
↑ 1.0  ┌──────╱─┐
       │     ╱ ←─── 밝은 영역을 더 밝게
       │   ╱
  0.5  │ ╱   ← 중간은 그대로
       │╱
       ╱   ←─── 어두운 영역을 더 어둡게
  0.0  └─────────→ 입력
```

- 어두운 끝을 내리고, 밝은 끝을 올리면 **대비 증가**.
- 본인이 "대비가 강한 진한 사진" 만들 때 쓰는 게 이것.

### 🔴 S 커브의 부작용 — 채도 증가

S 커브는 **대비만 올리는 게 아니다**. 색의 채도도 함께 올라간다. 톤이 진해지면서 빨강·파랑이 같이 진해지는 느낌.

이걸 원치 않을 때:

- HSL/Color Mixer에서 채도를 살짝 누르거나
- **Point Curve** 대신 **Parametric Curve**(슬라이더 4개)로 부드럽게 작업
- 또는 **Lab/HSL Curve**로 휘도만 만지기 (Lightroom Classic의 옵션)

## 페이드 룩 (Lifted Blacks)

```
출력
↑ 1.0  ┌────────┐
       │      ╱
       │    ╱
       │  ╱
  0.2  │╱      ←─── 검정 시작점을 0이 아니라 0.2로 올림
  0.0  └─────────→ 입력
```

검정의 출력값을 0이 아닌 0.15~0.25로 끌어올리면 **검정이 회색 빛**으로 바뀌면서 필름 같은 느낌. 인스타그램 초기 필터의 시그니처.

## RGB 채널별 커브 — 컬러 그레이딩의 입구

본인이 가끔 쓴다고 한 그것. 가장 강력한 도구.

각 채널을 개별로 조작하면 **영역별로 색을 입힐** 수 있다:

| 만지는 곳            | 효과                                       |
| -------------------- | ------------------------------------------ |
| Red 커브 어두운 끝 ↑ | 그림자에 빨강 추가 → 따뜻한 그림자         |
| Red 커브 어두운 끝 ↓ | 그림자에 시안 추가 → 차가운 그림자         |
| Blue 커브 밝은 끝 ↑  | 하이라이트에 파랑 추가 → 차가운 하이라이트 |
| Blue 커브 밝은 끝 ↓  | 하이라이트에 노랑 추가 → 따뜻한 하이라이트 |

**Teal & Orange 룩**(영화에서 가장 흔한 시네마틱 톤):

- Blue 커브 어두운 끝 ↑ (그림자에 파랑/시안)
- Red 커브 밝은 끝 ↑ (하이라이트에 주황)

## Color Grading 패널과의 차이

Lightroom 신버전의 **Color Grading**(예전 Split Toning) 패널이 RGB 커브의 일부를 GUI로 제공한다. 빠르지만 정밀도는 RGB 커브가 더 높다.

> **빠른 작업 = Color Grading, 정밀 작업 = RGB 톤 커브.**

## 실전 팁

- 처음엔 **Parametric Curve**(영역 슬라이더 4개)로 익숙해진 뒤, **Point Curve**로 넘어가기.
- 점은 **3~5개 이상 두지 않기**. 많을수록 부자연스러워지고 밴딩 위험.
- RGB 채널 커브는 **양쪽 끝을 살짝씩만**. 0.05 단위로도 큰 변화.
