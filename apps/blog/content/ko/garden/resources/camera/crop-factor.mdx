같은 초점거리 렌즈라도 **센서가 작으면 화각이 좁아진다**. 작은 센서가 풀프레임 이미지의 **가운데만 잘라낸(crop) 것**과 같기 때문.

## 핵심 그림

```
┌────────────────────────┐  ← 풀프레임이 보는 영역
│   ┌────────────┐       │
│   │  APS-C     │       │  ← 작은 센서는 가운데만 받음
│   │ ┌──────┐   │       │
│   │ │ m43  │   │       │
│   │ └──────┘   │       │
│   └────────────┘       │
└────────────────────────┘
```

같은 50mm 렌즈를 끼웠을 때:

| 센서              | 크롭팩터 | 50mm 렌즈가 만드는 화각 = FF 환산 |
| ----------------- | -------- | --------------------------------- |
| 풀프레임          | 1.0×     | 50mm                              |
| APS-C (소니/후지) | 1.5×     | **75mm** 같은 화각                |
| APS-C (캐논)      | 1.6×     | 80mm 같은 화각                    |
| m4/3              | 2.0×     | **100mm** 같은 화각               |

## 35mm 환산이란

다양한 센서 시스템을 **풀프레임(=35mm 필름) 기준 초점거리로 환산해서 비교**하는 방법.

> 실제 초점거리 × 크롭팩터 = 35mm 환산 초점거리

크롭바디로 25mm 단렌즈를 쓰면 → "35mm 환산 50mm" → 풀프레임 50mm와 같은 화각.

리뷰나 스펙시트에서 **"환산 24-72mm"** 같은 표기를 보면 이 값을 말하는 것.

## 흔한 오해

- **크롭이 화각을 좁힐 뿐, 진짜 망원이 되는 건 아니다.** 200mm 렌즈를 APS-C에 끼우면 **화각은 300mm처럼** 잡히지만, 차이는 **해상도(픽셀 수)에서의 손해**다. [[perspective-compression|원근 압축]]은 거리가 결정하므로 같은 거리에서 찍으면 압축감은 동일하다.
- **심도도 환산해야 한다.** 같은 화각으로 비교하려면 조리개도 크롭팩터로 보정. APS-C f/1.8 ≈ 풀프레임 f/2.7 정도의 배경 흐림.

## 풀프레임만 써왔다면

크롭팩터 1.0×로 살아온 셈이라 이 개념을 만날 일이 없다. 하지만:

- 시네마 카메라 (Super 35 ≈ 1.5×)
- 액션캠/드론
- 대부분의 스마트폰 (m4/3보다도 작은 센서)

이런 시스템 스펙을 읽을 때 환산 개념을 알아야 비교가 가능하다.
